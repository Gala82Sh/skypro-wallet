# Skypro Wallet

Приложение для учёта расходов с аналитикой и адаптивом под мобильные устройства.

## Технологии
- React 19
- React Router DOM 7
- CSS Modules
- LocalStorage для хранения данных

## Установка и запуск
```bash
git clone <ссылка на репозиторий>
cd skypro-wallet
npm install
npm run dev

> Все критерии выполнены.
