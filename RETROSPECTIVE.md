# Ретроспектива проекта SkyproWallet

## Что прошло хорошо
- Справилась с основным функционалом: авторизация, добавление расходов, аналитика, адаптив.
- Проект соответствует макету, адаптирован под мобильные устройства (375px).
- Код получился чистым и поддерживаемым.

## С какими трудностями столкнулись
- Долго настраивала синхронизацию данных между десктопом и мобильной версией через localStorage.
- Календарь на отдельной странице вызвал сложности с выбором периода и скроллом месяцев.
- Иногда возникали проблемы с редиректами в мобильной версии аналитики.

## Что можно улучшить в следующих проектах
- Больше тестировать мобильную вёрстку на ранних этапах.
- Заранее продумать структуру API-слоя (transactionsApi), чтобы не переписывать код несколько раз.
- Использовать Git-ветки для каждого крупного блока, чтобы проще было откатывать изменения.
